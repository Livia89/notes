# *Notes - ReactJS*

Create and management notes in text or audio using speech recognition API

## Requisites

- ViteJS + React;
- Typescript
- Speech Recognition;
  
## Setup

- Clone the repository;
- Install dependencies (`npm install`);
- Run application (`npm run dev`);
- Have a fun \o/
